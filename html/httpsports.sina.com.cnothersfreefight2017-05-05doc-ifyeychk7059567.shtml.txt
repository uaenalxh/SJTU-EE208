宝力高带队出战功夫全能技大奖赛 目标冲击三冠_武术散打_新浪竞技风暴_新浪网
2017-05-05
宝力高带队出战功夫全能技大奖赛 目标冲击三冠　　5月8-13日，“战无极”第二届全国体育院校功夫全能技大奖赛将在北京体育大学开战，本次赛事允许社会俱乐部报名参赛，宝力高国际搏击训练基地派出了“草原行者”韩国军、“草原野狼”马安定、“草原铁骑”包明明等六名队员出战。5月3日，我们来到宝力高国际搏击训练基地进行了探营，了解选手的备战情况，并对宝力高教练进行了专访。　　宝力高，蒙古族，1980年12月24日出生，内蒙古通辽人，绰号“草原骄子”，毕业于西安体育学院，是中国著名散打运动员，在当年叱咤风云的散打王“王中王”争夺战上，宝力高力克强手，加冕王中王至高荣誉，成为中国散打响当当的名片之一。此外他还是在“国王杯”上利用泰拳规则击倒泰拳王的第一个中国人，并且还是获得世界自由搏击联盟90公斤级WKA金腰带的第一个亚洲人，此外他曾在当年名噪一时的“中俄争霸赛”击败普京保镖、超级拳手须多维奇，获75KG冠军……可以说，“草原骄子”的荣誉数不胜数！　　在“中俄争霸赛”上宝力高击败俄罗斯超级拳手须多维奇获75KG冠军。在泰国本土击败泰拳高手，被称为“泰拳杀手”。2002年的散打王“王中王”争夺战，他力克强手，加冕王中王至高荣誉，此外他还是中国大陆首位WKA（世界自由搏击联盟）90KG金腰带拥有者。　　据宝力高教练介绍，宝力高国际搏击训练基地发展迅速，截止目前为止也就短短3年的时间。在这么短的时间里，在云南、河北、江苏、沈阳、内蒙几个地区也都开展了拳馆，同时，宝力高国际搏击训练基地也和学校合作，其中有1800人左右规模的学校就有三个。　　宝力高国际搏击训练基地本次有六人参赛，宝力高教练说这次参赛的目的有两个，一个是为了锻炼他们，另一个是为了支持这个比赛，支持这个国内原创品牌赛事。虽说是为了锻炼选手，但宝力高也表示，马安定、韩国军、包明明三位选手实力也都是不错的，有实力冲击冠军。　　从比赛报名的情况来说，有很多蒙古族选手，而且在各大赛场也经常看到很多蒙古族选手，小编发出疑问，为什么会有这么多蒙古族选手参赛，难道是因为他们天生就适合这项运动吗？对此，宝力高给出了自己的看法，他说：“可能是蒙古族的选手从小就开始骑马、摔跤，比较有天赋，也可能是因为这是新兴项目，大家都比较喜欢，所以越来越多的人加入这个项目。”　　在综合格斗项目上，中国运动员和国外运动员还是有一定的差距的，目前中国的很多综合格斗运动员都是散打、摔跤等项目转型过来的，比赛中还是会以这些项目为主，针对这一现象，宝力高表示：“中国的运动员要跟着市场去调整，目前中国的综合格斗不仅要在站立这些基本技术上增加突破点，也要增加地面的技术，中国运动员最欠缺的就是在地面技术的逃脱上。例如UFC，他们的运动员地面技优秀了，他们就多加强站立技术。”　　作为老一代运动员，宝力高的小目标就是多培养几个像李景亮这样的在散打或者是综合格斗领域有一定水平的运动员，这样说是因为当初李景亮就是跟随他从新疆走出来的。同时，作为年轻运动员的前辈，他也给年轻的运动员提出了一些建议，他说：“现在的年轻运动想在这个项目中取得一定的成就，首先就是要先把自己的技能练好，不要过多的看重名利。其次，就是准备好自己的每一场比赛，接受每一场比赛的考验，做好自己，打出水平。每个人都有机会，看谁能付出。”　　正因为越来越多的人加入，搏击圈如今非常的火爆，针对是走出去还是引进来的问题也是众说纷纷。宝力高也说：”中国搏击虽然火爆，但是并没有形成规模，中国搏击不仅要引进来还要走出去，走出去见识一下别的国家的训练方法，吸收他们更高、更快，正能量的东西，结合我们传统的训练方法，来提高我们的水平。同样的引进来他们好的运动员、教练员，也是相应的道理。”　　最后，谈到中国的运动员最好的状态，最好的胜利是否是打到UFC，宝力高表示不认同，他说：“UFC是一个项目、一个组织、一个商业赛事而已，他的成功只是宣传的比较好。例如我们的全运会，我们不能说他水平低，只能说是UFC的宣传比我们的散打宣传的好而已。而且UFC都三十多年了，中国的商业赛事最多的也就十几年，中国人那么多，以后我们肯定会有属于我们国家自己的品牌，未必要去打UFC，到时候让国外选手来中国打我们的比赛。而且国外运动员的实力也不一定好，也有很多人倒在我们中国运动员的拳头下。”聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
