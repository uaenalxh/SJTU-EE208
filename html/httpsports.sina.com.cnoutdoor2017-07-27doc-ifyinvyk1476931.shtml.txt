情侣对飙 卡拉宝DM商场速降即将引爆长沙_户外频道_新浪竞技风暴_新浪网
2017-07-27
情侣对飙 卡拉宝DM商场速降即将引爆长沙　　经过一个多月的休整，2017卡拉宝DM极限全国巡回赛本周末再度重磅来袭！7月29至30日，中外速降精英将齐聚长沙开福万达广场，为您带来一场激动人心的视觉盛宴！　　商场速降（Down Mall，简称DM）就是把本该在原始深山茂林进行的极限越野下坡赛搬到最具现代生活气息的城市商场中进行。　　2017卡拉宝DM极限全国巡回赛前两站的精彩比赛让这个赛事在业界得到了越来越多的认可，除了众多参加过前两站的选手继续追随之外，也有不少新晋车手慕名前来参赛。报名本站比赛的选手可谓看点多多。其中，有获奖无数，衣锦还乡的 “速降小王子” 20岁湖南天才少年唐梦麒；还有常颜、鄂子才、麻煦堯、舒卓凡、周小焜五名长沙本地小鲜肉勇敢参赛，这些年仅16岁的选手是否给我们带来惊喜呢？　　俗话说，巾帼不让须眉，危杨洋和韦健明这两位巾帼豪杰的参与，将为长沙站的比赛增添一抹亮色，特别是被称为“广西一姐”的危杨洋，本次将携男友尹铂涵一同参赛，这对情侣骑手能否在本站同创佳绩，值得关注。除了国内选手之外，有多名外国骑手也报名前来挑战，让人对这一站比赛的精彩程度充满了期待。　　除了参赛选手有故事之外，主办方在本站比赛的赛程设置上也下足了功夫。除了常规的DM极限竞速赛之外，还增设了飞跃跳远赛的环节，车手通过一段距离的加速，飞跃跳台，以跳跃的距离排定这个项目的名次。届时，国内外一众高手各显其能，呈现最刺激眼球的视觉盛宴，势必让您大饱眼福。另外，长沙的观众还可以现场报名和参赛选手进行PK，亲身体验这项运动的独特魅力。　　喜欢就来吧！本周六、日，长沙开福万达广场，我们不见不散！　　　　卡拉宝DM极限全国赛长沙站赛事日　　程速降排位赛：7月29日14：45-16：30　　飞跃表演赛：7月29日16：30-17：30　　速降决赛：7月30日15：15-16：30　　观众挑战赛：7月30日17：15-18：00　　2017卡拉宝极限全国巡回赛共设十场比赛，赛事从6月进行至11月，赛区覆盖全国九大城市，包括广州、东莞、长沙、北京、成都、杭州、南京、南宁、深圳。整个赛事由泰国原装进口、不含人工咖啡因的卡拉宝维生素果味饮料独家冠名赞助。每场比赛均接受公开报名，详情请关注赛会官方公众号。聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
