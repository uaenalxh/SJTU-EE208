南安普顿主帅：想留下范迪克 但我没法决定这事_英格兰_新浪竞技风暴_新浪网
2017-12-18
南安普顿主帅：想留下范迪克 但我没法决定这事　　南安普顿主帅佩莱格里诺承认范迪克不在自己的掌控范围内，荷兰国脚随时都可能离队。　　26岁的范迪克将在冬天成为众多豪门追逐的球星，据报道，曼城、利物浦、切尔西都对荷兰人非常感兴趣。　　圣徒主帅佩莱格里诺在接受采访时说道：“我控制不了市场，我不是球员的主人。我能够把意见反馈给技术总监，然后交到主席。之后董事会会决定一切，这对团队来说是最好的。”　　“再一次强调，我现在无法控制转会市场，我建议留下俱乐部最好的球员。俱乐部一开始就很清楚，我也有相同的想法。”　　南安普顿目前排在联赛第12，在12月底之前球队还将面临艰难的客场之旅。　　（来自 肆客足球）聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
