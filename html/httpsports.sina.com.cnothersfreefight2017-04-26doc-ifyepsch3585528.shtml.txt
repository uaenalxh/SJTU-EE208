首体型男再战体院大奖赛 称得张开印指导受益良多_武术散打_新浪竞技风暴_新浪网
2017-04-26
首体型男再战体院大奖赛 称得张开印指导受益良多　　有“京西双少”之称的“京西大少”杨宇和“京西二少”曹政近来在CKF赛场打的是风生水起，连克多位名将，在江湖上威震一方。　　5月8-13日，“京西二少”曹政将代表首都体育学院参加“战无极”第二届全国体育院校功夫全能技大奖赛，这已经是曹政第二次出战该赛事了，在2015年，曹政获得了首届比赛65公斤级的第三名，本次比赛曹政将参加70公斤级的较量。　　现在在读大四的曹政目前学业上主要在专心做毕业论文，对于参加本次比赛，曹政说：“参加第一届比赛时得到了很多校友的支持，那次最终获得了第三，有点遗憾，我现在马上就要毕业了，非常珍惜这次能够为校争光的机会，希望这次能够取得更好的成绩。”　　从记者目前拿到的名单来看，本届大赛70公斤级这个级别好手云集，“石头人”陈磊、“小钢炮”胡景辉、哈斯特尔-阿克木江、沙依兰、赵依昆、张蓬飞等16位拳手报名。看到这样的阵容，曹政首先开玩笑说：“我准备退赛了！”　　本届比赛，首都体育学院有六名队员报名，相比第一届比赛翻了一倍。在报名的六人中，曹政是唯一一位有多场商业赛经验的队员。曹政说：“70公斤级这16位报名队员大部分都有所了解，实力都很强，好在这次我没有降重压力，将会以最饱满的状态出战。”　　目前，曹政在崔建功、张开印两位教练的指导下，每天都在积极备战。拿到过诸多荣誉的“威风少侠”张开印更是给了曹政很多有益的指导。曹政说：“印哥在训练时对我们非常严格，有一定的心理压力，但是对我们打比赛是好的。”聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
