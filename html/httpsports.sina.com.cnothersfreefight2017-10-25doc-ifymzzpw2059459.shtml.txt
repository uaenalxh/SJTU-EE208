他将超越柳海龙? 新领军者能否扛起散打大旗　　曾经盛极一时的“中国武术散打王争霸赛”在中国功夫和武术散打的历史上，都具有着里程碑般的重大意义。而作为“散打王争霸赛”最重要的名片，世界超级散打王柳海龙则是那个散打最辉煌时代的代表人物。　　世易时移，老一代散打王们均已先后退役，而“散打王争霸赛”也停办十年有余。如今新散打王赛事即将再次复活，2017年世界超级散打王争霸赛也将于10月28日19：00鸣锣开战，一代新散打王呼之欲出。我们不禁好奇，谁又能接过上一代超级散打王柳海龙的大旗呢？　　付高峰，散打冠军大满贯、中国散打国家队队长，恐怕是业内人士脑海中浮现的第一个名字。不久前，付高峰在接受媒体采访时也曾坦言：师兄柳海龙一直是自己的偶像，但是在他们相处的时候，付高峰仍当面向师兄许下诺言，总有一天他将超越自己的偶像，成为新一代的散打领军人物。　　实际上，现任国家散打队队长的付高峰，早已可以称作是新一代散打的领军人物了。因为，自老“散打王争霸赛”停办以后，散打最高水平赛事当属全运会、全国散打锦标赛/冠军赛和世界散打锦标赛/世界杯了。而付高峰自2011年至今，已连续斩获了六次全国冠军，还多次夺取全运会和散打世界杯冠军，是不折不扣的散打冠军大满贯得主。而作为现役散打国家队队长，在各大体育媒体上，付高峰也总是以散打一哥的形象出现。　　和巅峰时期的柳海龙相比，付高峰也许唯一还欠缺一些的，就是职业赛场上的经典战例较少。这主要是因为作为一名体制内运动员，付高峰并不能随心所欲的参加商业赛事，因此在职业赛场中亮相机会不多，高光时刻自然不多。不过少数的几次亮相中，仍然可以让拳迷们感受到散打一哥的恐怖实力。30秒KO加拿大名将麦奇·福勒；一回合KO日本悍将相良正幸；两胜WKN世界泰拳冠军、比利时名将阿卡·马提瓦；5回合大战，击败WBC、WMC、WKN泰拳世界金腰带，Glory中量级冠军、世界泰拳一哥莱文；在为数不多的职业赛场的亮相次数中，付高峰在自己并不熟悉的规则下，仍旧证明了他早已跻身世界顶一流好手的行列。　　如今，散打王王者归来，“世界超级散打王争霸赛”已正式定下归期，届时，将有更多的职业比赛以及对阵国外搏击高手的机会。因此，付高峰也赢来了证明自己的最佳时机，他必将用更多职业赛的胜利来丰富自己的荣誉墙，告诉世人谁会扛起散打领军大旗。随着新散打王赛事的逐步开展，精彩比赛必然会越来越多，让我们拭目以待，静候这位散打一哥的优异表现吧。聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
