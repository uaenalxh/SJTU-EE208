它是中国格斗最早的脊梁 柳海龙从这里走向世界　　中国武术散打王争霸赛曾是中国搏击的最高代表，巅峰时期曾造就出中国最早的一批搏击明星，如柳海龙、宝力高、边茂富、苑玉宝等等，均威名远扬。　　其中最为著名的要数“柳腿劈挂”，一个曾经享誉世界搏击界的名号。连续四年全国冠军、五次散打王称号、两次散打世界杯冠军、世界自由搏击IKF金腰带、超级散打王，一系列的头衔和荣誉的光环，让人无法直视，而它们的拥有者就是中国散打代表人物柳海龙。他是中国散打最辉煌时期的领军人物，是中国散打最鲜明的一面旗帜。　　更让中国拳迷推崇备至的是，柳海龙辉煌的搏击生涯，并不仅仅局限在国内的散打规则赛事。在散打王争霸赛：中外对抗赛单元中，柳海龙在与美国、俄罗斯、泰国、日本、伊朗等世界各格斗强国高手的对抗中，都表现出了统治级的王者风范。在散打王中泰职业拳王对抗赛中两胜泰国泰拳王，甚至差点将泰拳王扔出拳台；在散打王中美自由搏击对抗赛中击败美国选手其并以标志性的劈挂腿将对手脸部肌肉砸裂，血流不止；以绝对优势大比分战胜日本KO之王伊贺弘治，捍卫了中国武术荣誉；等等神级表演，直至今时，仍是无数老拳迷心中永恒的经典。　　相辅相成，正是由于“中国武术散打对抗赛”这样的高水平赛事，才培养出了柳海龙、宝力高、边茂富这样一大批高水平职业搏击选手、明星。也正是散打王们在国际国内赛场上的精彩表现，让中国散打成为了世界上最著名的站立格斗术之一，甚至一度让人们忘记李小龙，而成为了中国功夫的代名词。散打王争霸赛成为了中外武术交流的最高平台，而散打王们也成为了中国武术和国外格斗界对抗的唯一主力。散打王争霸赛的一众散打王们在世界擂台上与各大搏击强国高手们的不屈不挠、热血拼搏，铸就了中国格斗最早的钢铁脊梁。即便在散打王停办十多年之后，现在中国最高水平的职业搏击选手的身上也仍旧或多或少的烙有散打的印记。　　近日一个好消息传来，不久后，散打王即将复活！2017年世界超级散打王争霸赛，将于10月28日在国家奥林匹克体育中心体育馆鸣锣开战。新时代、新规则、新面孔，新的散打王争霸赛能否再次造就一批威名赫赫的王者，令人十分期待！聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
