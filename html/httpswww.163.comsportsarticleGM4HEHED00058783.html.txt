2021南昌马拉松赛道丈量工作已完成|琼斯|赣江|南昌国际体育中心_网易体育
2021-10-12
2021南昌马拉松赛道丈量工作已完成用微信扫码二维码分享至好友和朋友圈网易体育10月12日报道：2021赛道于10月12日丈量完成，丈量工作自10月11日21:30开始，次日清晨结束。南昌，别称“天下英雄城”，南昌马拉松也被誉为“英雄马”，2021英雄马将于11月6日鸣枪起跑。本次赛事由中国田径协会、江西省体育局、南昌市人民政府主办，南昌市体育局、东湖区人民政府、西湖区人民政府、红谷滩区人民政府、江西省田径运动协会承办，智美体育文化（浙江）有限公司运营推广。本次赛事被列为“奔跑中国”马拉松系列赛，中央电视台体育频道（CCTV5）将对赛事进行直播。马拉松赛事的赛道丈量是按照世界田联规定进行，丈量工作在公安、交警部门的协助下，以自行车为交通工具，以记录仪的数据为支撑，模拟运动员最短的跑步路线，每一公里停车一次，对记录数据进行计算，并对公里数位置进行标记、定点、描述。根据丈量数据，对赛道路况进行判断，对赛道细节进行调整与优化。琼斯记录仪的原理是先校准、测量好车轮滚动一圈的精确周长，然后通过在自行车上固定圈数计数器计算所走距离。据悉，担任本次赛道测量的是世界田联B级丈量员胡新民。据了解，今年的英雄马赛道进行了全方位的优化，起点在彰显南昌“天下英雄城”红色底蕴、具有恢弘历史的八一广场，终点在南昌国际体育中心4号门。本届赛道有四大升级：赛道更为平缓，适合PB；减少了转弯与绕行，整体比赛体验更加顺畅；大道南北通畅，地标罗列，风光绝美；终点再回南昌国体。从组委会发布的路线看，2021英雄马赛道将有助于跑者创造更好的个人成绩。具体路线：1.马拉松路线起点：八一大道（江西省展览中心门前）→（左转）阳明路 → 八一大桥 → 庐山南大道 →（右转）丰和北大道 →（右转）香江路 →（左转逆行）赣江北大道  → 长江路 → （折返逆行）赣江北大道 → 赣江中大道 → 赣江南大道 → 吉安街路口南（折返逆行）赣江南大道 → （左转逆行）北龙蟠街 →（右转逆行）九龙大道 → 南昌绿地国际博览中心（折返逆行）→ 九龙大道 →（左转逆行）北龙蟠街 →（左转逆行）赣江南大道→（左转） → 南昌国际体育中心4号门（终点）2.半程马拉松路线起点：八一大道（江西省展览中心门前）→（左转）阳明路 → 八一大桥 → 庐山南大道 →（右转）丰和北大道 →（右转）香江路 →（左转逆行）赣江北大道  → 长江路 → （折返逆行）赣江北大道  → 赣江中大道 → 赣江南大道 → （右转） → 南昌国际体育中心4号门（终点）3.迷你欢乐跑/家庭跑路线起点：八一大道（江西省展览中心门前）→（左转）阳明路 →八一大桥→庐山南大道→铜锣湾（终点）据了解，诞生于2016年的南昌马拉松，首届举办就因美丽的赛道、出色的竞赛组织、贴心的赛事服务在当年被评为“中国田径协会铜牌赛事”，近年来英雄马参赛规模不断扩大，赛事品质不断提高，国际、国内赛事综合排名均有所提升， 2021年，南昌马拉松被评为世界田联精英标牌赛事。想要了解更多南昌马拉松信息，大家可登录官方网站或关注南昌马拉松公众号。乐糕民生资讯 2022-12-06 03:13:40直播吧 2022-12-06 04:03:29体坛无名 2022-12-06 04:02:40老胡聊球 2022-12-06 09:31:59网易体育 2022-12-06 06:22:58新华社客户端 2022-12-05 22:01:04农人小袁 2022-12-06 06:49:03世界图影 2022-12-06 01:53:44界面新闻 2022-12-05 17:01:31老王谈近现代 2022-12-06 03:41:14小火柴视频 2022-12-06 11:31:47网易体育 2022-12-06 10:24:49网易体育 2022-12-06 08:50:48央视新闻客户端 2022-12-06 07:25:46第一财经 2022-12-06 11:38:29网易体育 2022-12-06 13:35:00中国新闻周刊 2022-12-05 19:49:08网易体育 2022-12-06 12:13:18
