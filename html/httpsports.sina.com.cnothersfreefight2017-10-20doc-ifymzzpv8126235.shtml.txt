散打王争霸赛重装回归 柳海龙等名宿将到场助阵_武术散打_新浪竞技风暴_新浪网
2017-10-20
散打王争霸赛重装回归 柳海龙等名宿将到场助阵　　中国武术协会、国武时代国际文化传媒（北京）有限公司联手打造的2017年世界超级散打王争霸赛（2017 WKA Kungfu King of the World Championship）将于10月28日在北京国家奥林匹克体育中心体育馆激情上演。 2000年诞生的“中国武术散打王争霸赛”是中国武术散打的标志性符号，是中国武术散打的第一品牌，也是世界上独一无二的中国民族赛事品牌。“散打王”曾经是一代人澎湃的青春记忆，2005年后淡出观众视野，成为人们心中无法磨灭的美好印记。     　　12年，在经历了一个轮回的蛰伏后，“散打王”破茧重生，重装回归。     　　武术是中国的传统文化，散打是中华武术的精华。2017年世界超级散打王争霸赛就是要在各种搏击赛事的大潮中竖起散打的大旗，搭建规范平台，引领国际搏击市场。赛事主办方希望通过打造散打的民族赛事品牌，可以让大家关注中国武术，喜爱武术散打，弘扬尚武精神。     　　回归之后的散打王，相较以前，不变的是将继续传承中国的传统文化，弘扬民族精神。而“大健康、大体育、大娱乐”背景下新的时代也赋予了回归后的散打王赛事新的特质：贴近大众、贴近市场、贴近生活，助力全民健身，为全面建成小康社会，实现中国梦的宏伟目标做出贡献。     　　10月28日晚7点，代表当今中国散打最高水平的中国选手将联手挑战伊朗、埃及、印度、黎巴嫩等国家的世界散打顶级高手，争夺全球散打的最高荣誉——世界超级散打王。 本次比赛共分为7个级别，由于中国武术协会的直接参与，赛事规格和选手水平均为顶级。　　60公斤级中方出场选手为今年新科全运会冠军孔洪星，此前，孔洪星已经包揽60公斤级散打比赛所有金牌。他的对手是2016年世界杯武术散打比赛银牌得主，来自印度的苏里亚·巴努·帕塔·辛格。     　　65公斤级中方选手张腾，重拳、重腿，技术风格硬朗朴实，是“硬派散打”领军人物，刚刚夺得第14届世界锦标赛金牌。对手是来自伊朗的埃尔方·阿汉加兰，曾经夺得2016年亚洲武术锦标赛金牌，是今年伊斯兰团结运动会、世界大学生运动会双料冠军。     　　70公斤级中方选手冯杰和75公斤级中方选手许永均为去年亚锦赛冠军得主。冯杰多次在各类职业比赛中战胜国外选手，他的拳法凌厉凶猛，有“最硬散打人”之称。冯杰和许永的对手分别为埃及的艾曼·加拉尔·法特希和黎巴嫩的贝希尔·亚明，前者在刚刚结束的世锦赛中位居第二，后者曾经获得过散打世界杯第三名。     　　80公斤级中方选手付高峰曾经痛快打败被人们称为“中国散打噩梦”的俄罗斯高手穆斯里穆，被人看好为超级散打王柳海龙的“接班人”。他的对手是来自韩国的仁川亚运会冠军得主金明真。　　90公斤级中方选手王文忠是去年亚锦赛和散打世界杯的双料冠军，在一个月前的天津全运会中却惨遭高鞭腿KO，正欲找寻机会一雪前耻，证明自己。他的对手是2017年世界武术锦标赛金牌获得者、伊朗选手米拉德·阿雷菲·曼格姆。     　　“中国散打重量级第一人”叶翔代表中国队参加90公斤以上级比赛，叶翔是2017年第14届世界武术锦标赛90公斤以上冠军得主。他的对手是来自埃及的世界武术锦标赛金牌得主莫塔斯·雷迪。     　　当晚比赛还包括一场女子表演赛，中方选手为以充沛体力著称的“拳坛一姐”王聪。王聪的对手为22岁乌克兰姑娘纳塔莉亚，曾获得跆拳道、泰拳世界冠军。     　　七场比赛，一场表演赛，场场恶仗，高手对决，必将碰撞出激情火花。     　　比赛当晚，中国武术超级散打王柳海龙，散打王中王宝力高，散打王赵子龙、杨晓靖、郑裕蒿、李杰、那顺格日勒等往届散打王霸主也将荣光现身。群星闪耀，巅峰对决，多媒体集群直播，上演现实版“战狼”，展示中国力量与智慧，共同见证2017年世界超级散打王的诞生。 聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
