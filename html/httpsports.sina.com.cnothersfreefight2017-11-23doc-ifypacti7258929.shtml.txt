穆斯里穆谈偶像柳海龙 武亚楠要做UFC花木兰_武术散打_新浪竞技风暴_新浪网
2017-11-23
穆斯里穆谈偶像柳海龙 武亚楠要做UFC花木兰　　11月23日，在上海浦东嘉里大酒店，UFC上海站举行了赛前的公开训练和发布会。　　新闻发布会是在一个宴会厅里进行的，11场参赛的22名的拳手中，只有6人得到了官方的采访安排。　　其中包括比赛的主赛拳手比斯平和盖斯特鲁姆，而另外四名拳手的安排很有意思，分别是中国本土的李景亮、王冠和武亚楠，另一个人是在中国很出名的散打沙皇、拿到过散打世锦赛冠军的穆斯里穆。　　发布会是从12时40开始的，首个出场的是外国练散打的高手代表穆斯里穆。也许上海—中国是穆斯里穆入选、获得UFC合同的一个原因吧。　　“中国通”穆斯里穆在媒体见面会上回答问题的时候，很会拉进自己和中国粉丝的距离，他说：“我很小的时候就看柳海龙的比赛视频，长大后我开始练散打，我非常希望可以像他那样比赛，我后来还和他一起进行过训练。在UFC擂台上，有很多的站立高手，我也希望我可以打出散打的技术比如回旋踢来，展示散打的风采。”　　这次的比赛有两名中国女选手入选战列，按照赛程武亚楠应该成为第一个进入八角笼的中国女性。　　“这是压力也是动力吧。”武亚楠很大声地回答着记者们的提问。　　一名外媒记者问她：“在不同文化里，对于女性练MMA有不同的看法，你能得到自己家人的支持么？”　　武亚楠用一个具有世界认识的名字来解释了中国女性的战斗精神，她说：“中国文化也有花木兰，我的家人父母一开始也不赞成我做这个事情，不过在我拿到UFC　　的合同后，他们理解和支持我、鼓励我了，因为UFC是这个行业的殿堂，我会努力打下去，只有战胜冠军，自己才是冠军。”（周超）聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
