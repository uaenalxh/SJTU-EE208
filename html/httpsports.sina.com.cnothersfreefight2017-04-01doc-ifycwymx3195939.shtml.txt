功夫全能技大奖赛首届冠军 巴特尔张开印或带队_武术散打_新浪竞技风暴_新浪网
2017-04-01
功夫全能技大奖赛首届冠军 巴特尔张开印或带队　　5月8-13日，第二届全国功夫全能技大奖赛将在北京体育大学体育馆火爆上演，全国搏击精英将向功夫至尊王位发起冲击。　　全国功夫全能技大奖赛的前身是2015年全国体育院校功夫全能技大奖赛，当时“嗜血猎手”刘晓洋、“断流天王”逯建波、“草原苍狼”巴特尔、“威风少侠”张开印、“塞外豪侠”青格乐分获五个级别冠军。可惜本次比赛，这五位冠军因为不同原因无缘本次比赛，提前无缘卫冕。　　刘晓洋　　“嗜血猎手”刘晓洋是60公斤级的卫冕冠军，回忆2015年的冠军之路，刘晓洋记忆犹新，“第一场比赛第一局手就打骨折了，到最后一场获胜变成了熊猫眼，就像昨天发生的，很深刻的一次比赛经历。”　　那次比赛之后刘晓洋养伤半年多，复出之后获得“拳王统一战”冠军，成为博克森功夫全能技战队正式签约队员，享受18万/年的保底薪水，由于在2016年年末功夫世界杯热身赛中受伤，至今还没有痊愈，无缘参加本次全国功夫全能技大奖赛。　　逯建波　　“断流天王”逯建波是66公斤级的卫冕冠军，当年酷酷的出场方式给众多拳迷留下了深刻的印象。逯建波表示我已经拿过一次冠军了，证明了自己的实力，西安体育学院还有很多非常出色的师弟，应该给师弟们更多的机会。　　巴特尔　　“草原苍狼”巴特尔是70公斤级的卫冕冠军，当时夺冠后巴特尔感慨现在的年轻人拼劲不足，并没有给他造成足够的威胁，希望年轻人赶快成长起来，扛起中国搏击的重任。巴特尔在接受采访时表示他不会以队员的身份来参加5月的比赛了，但是有可能以教练的身份出征。　　张开印　　“威风少侠”张开印是77公斤级的卫冕冠军，当时张开印还是北京体育大学的研究生，现在张开印已经到首都体育学院担任教师，张开印在接受采访时明确表示会以教练的身份带队参加本次比赛。　　青格乐　　“塞外豪侠”青格乐是85公斤级的卫冕冠军，当年三场比赛，他以一个KO，两个TKO轻松夺冠。在今年1月的功夫世界杯比赛中，青格乐战胜了法国“竞争者”金腰带得主，可惜他在比赛后查出肋骨骨折。近几个月青格乐始终处于养伤之中，没有进行系统训练。青格乐在接受采访时表示很遗憾无缘这次全国功夫全能技大奖赛。　　五位卫冕冠军全部无缘本次比赛，这也意味着每个级别都将产生新的冠军，谁能成为新的王者呢？让我们拭目以待。中国互联网电视将全程播出本次赛事，敬请关注！聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
