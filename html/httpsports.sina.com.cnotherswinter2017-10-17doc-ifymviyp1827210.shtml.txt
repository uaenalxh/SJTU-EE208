中国体育报：冰雪进校园尚需资金支持　　退役后在坡面障碍国家队担任领队的冬奥会自由式滑雪男子空中技巧冠军韩晓鹏日前来到北京大兴国家教育行政学院附属实验学校，出席由北京市大兴区教委主办、北京冰雪梦体育文化有限公司协办的“大兴区中小学冰雪运动进校园专题讲座暨仿真冰旱雪体验活动”，向孩子们分享运动训练经验和冬奥会夺冠感受。　　活动现场，在学校操场集中展示了仿真冰、室内模拟滑雪机、仿真旱雪场、仿真冰壶场地等，便于与会学校代表近距离了解场地设施情况。来自国家教育行政学院附属实验学校和庞各庄中学冰上运动社团的学生们进行了现场演示。在北京市教委提出冰雪进校园的政策下，此次活动尤为具有实际的服务意义，全区的87所学校全部出席，希望可以在场地建设、师资提供方面得到新的信息和解决方案。　　据了解，北京市多所学校计划或已经开展冰雪活动，但在实际操作中遇到不少问题和阻力。大兴区庞各庄中学是北京市第一批52所冰雪特色校之一，副校长林建立介绍，学校现已成立滑冰、滑雪社团，并参加了相关比赛取得了良好的成绩，学校有充足的教学场地、体育馆等，通过市场了解到仿真冰场和旱雪装备，希望上级领导能给予更多的资金支持，建设校内防冰场和旱雪场，这样不仅可以满足社团活动更好地开展，同时还能开展校本课程，让更多的孩子参与进来，培养更多的冰雪人才。　　北京市润丰学校也是北京市第一批52所冰雪特色校之一，开展冰雪运动较早。据校长卓立介绍，学校在2016年5月成立冰球队，多次参加北京市中小学生校际冰球联赛，社团活动方式主要是由学校统一组织在校外滑冰、滑雪运营场馆以购买服务形式开展冰雪运动，此种形式并不能满足学生日常的训练，学校计划2018年在校内搭建仿真冰场、购买社会教练服务形式，使社团学生日常的训练次数能得到保证，并且可以让全校学生参与到冰雪运动中来，打造具有冰雪特色的校园文化。　　延庆区太平庄中心小学冰雪项目建设已经成型，今年参加了北京市首届中小学冬季运动会并获得了四金四银四铜的优异成绩，延庆选拔区速滑队，四十人中有该校学生十四人。校长丁建培希望上级领导对校园冰雪项目给予更多重视，如果教委给投资建四季冰场，对孩子们是一个好事。　　国家教育行政学院附属实验学校计划将冰雪运动纳入学校体育课程体系中，校长李淑新提到，目前学校具备建设仿真冰场的条件，计划搭建仿真冰场，为学校开展冰雪运动提供基础场地，但学校在冰雪师资方面存在短缺，需要社会上相关服务单位给予支持，将冰雪运动融入学校的办学理念中。　　场地、课程、师资、资金是目前学校开展冰雪运动面临的最主要的困难。学校希望各级教委加大专项资金扶持力度和服务体系完善程度。大兴区教委体美科科长李克明透露，大兴区目前已经与区内经营性冰场雪场合作，将冰雪课程列入全区实践大课堂的内容，各学校可以根据学生的需求和学校课外课程安排选用。明年计划将相关课程逐步列入校本课程，目前已经有10多所学校提出铺设冰场及相关辅助训练设施的需求，教委将统一列入建设计划。　　多年来致力于为学校课外体育提供解决方案的北京冰雪梦公司总经理王红生认为，在推进冰雪进校园的过程中，应发挥多方力量，鼓励政府、社会组织、高校参与“冰雪运动进校园”。政府进行政策引导，通过购买服务的方式带动企业的参与，为学校提供专业指导、实践机会。同时，高校以学术支持形式进行宣讲和研究，将更多学术成果服务于冰雪及奥林匹克的推广。他建议，加大经费投入冰雪项目与管理服务体系，搭建“互联网+冰雪”教育服务平台，整合所有上游产业资源包括政府、协会、赛事组织、冰场、雪场、俱乐部、培训机构、装备企业、院校等，通过平台以政府购买服务的形式为百万青少年上冰雪提供专业标准的服务。  　　中国体育报聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
