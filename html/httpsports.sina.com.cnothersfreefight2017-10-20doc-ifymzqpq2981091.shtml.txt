柳海龙付高峰现身沧州 两代散打传奇助力英雄魂　　46号文后，中国搏击赛事如雨后春笋不断涌现。职业搏击迅猛发展，“泰国风”、“国际风”一阵阵刮过中国搏击产业，有人高呼期待中国搏击赛场的“中国风”。“宝华国际英雄魂”应运而生，在自由搏击、MMA为主导激烈的市场竞争中，通过差异化的赛事设计“打”出中国特色，令中国散打绽放光芒。　　曾KO泰国皇家拳师蓝桑坤的许振光、蒙古族散打名将“草原黑狼”包舍日古冷、“死神”方便、“降龙罗汉”白近斌、“沧海一剑”曹鲁健、“插翅虎”王伟豪等散打名将都曾在“宝华国际英雄魂”擂台上展现实力，为散打在职业搏击赛场上的竞争力正名。　　2017年11月11日，“宝华国际英雄魂”在河北沧州体育馆擂响战鼓，作为系列赛事的第十站，本次比赛不仅由“铁血震乾坤”许振光领衔中国战队，更迎来了两位散打领域的天王巨星--“中国散打王中王”柳海龙以及国家散打队队长付高峰。　　17年前，2000年中国武术散打王争霸赛总决赛中，19岁的柳海龙一晚连胜三场，跨级别击败100公斤的任彦兵，荣登中国史上第一个散打王的宝座。此后柳海龙蝉联王座，两度取得“超级散打王”荣誉，先后获得散打全运会、散打世界杯和世锦赛冠军。在中外对抗赛中，柳海龙证明了散打完全有能力立足世界武坛。2001年首届中泰对抗赛事，柳海龙两次将泰国拳王哥邦贵扔下擂台，打破泰拳五百年不败神话。2002年中泰对抗赛中，柳海龙再次作为压轴主将出场击败泰拳冠军江盖。2003年12月7日击败美国自由搏击冠军佛吉亚诺获得世界自由搏击IKF金腰带，并且2次击败空手道强将，尽现“龙威”。柳海龙展现出散打“以快打慢、以巧破力、以弱胜强、灵巧变化、技法丰富”的特点，攻防转换灵活，大开大合的披挂、力拔千斤的摔法、左右腾空连续双飞腿、甚至是不可思议的剪刀腿都信手拈来，让人领略了中华武术的博大精深。　　柳海龙以王者身份退役，随后付高峰成为了下一代的散打巨星接班人。作为散打国家队队长，付高峰连续6年霸占全国冠军宝座，曾连克散打冠军“降龙罗汉”白近斌和“草原苍狼”巴特尔，被认为是国内散打最强。在中外对抗中，付高峰击败史上最强外籍散打冠军“散打沙皇”穆斯里穆，并挑落当时世界排名第一的俄罗斯名将莱文，捍卫中国武术荣誉。由于和柳海龙都是80公斤级别，也都是山东人，所以付高峰也被誉为“柳海龙接班人”。　　柳海龙和付高峰两代散打巨星作为“宝华国际英雄魂”的嘉宾，不仅是对于“宝华国际英雄魂”赛事品牌的认可，同时也让拳迷感受到散打精神和技术的传承。英雄魂创始人秦贞伟也是散打运动员出身，对散打有着深厚的情感。“很荣幸柳海龙和付高峰能参与到本次英雄魂比赛中。中国搏击有了好的机遇，散打迎来再一次腾飞的机会。在职业赛场上，我们优秀的散打运动员同样能在MMA 和自由搏击的国际规则下对抗强敌。通过交流和切磋，中国运动员也在不断进步。以前大家都认为散打只会打点，但本次领衔出战的许振光多次重拳秒杀KO ，证明了散打同样有KO 能力。英雄魂会为中国高手提供国际级的舞台，弘扬尚武精神，展现出中国功夫魅力！”11月11日，散打高手汇集沧州，精彩即将上演！聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
