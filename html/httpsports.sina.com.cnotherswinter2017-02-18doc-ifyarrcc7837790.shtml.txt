女子滑雪时撞树受伤 雪场防护网常被人为破坏 _冰雪_新浪竞技风暴_新浪网
2017-02-18
女子滑雪时撞树受伤 雪场防护网常被人为破坏 　　法制晚报讯（记者 张群琛）近日在崇礼万龙和太舞两个雪场发生的滑雪者意外身亡事故让雪场安全再度引发人们的关注。随着气温回升，崇礼目前 迎来雪季末的客流高峰，法晚记者从雪友处了解到，一名女性滑雪者日前在万龙雪场滑雪期间因不慎撞上了雪道旁边的 树木受伤，被救援队带下山紧急送医。　　上月，一名北大女生在万龙雪场滑雪时，就是由于滑雪时速度过快撞到了雪道旁边的树上不幸身亡的。记者了解到，目前崇礼的六家雪场均已加装防护网。　　雪场意外 滑雪者可能速度过快 撞树后倒地不起　　雪友向记者提供的定位在万龙滑雪场的截图显示，几名雪场的救援人员正在为一名倒地不起的女性滑雪者提供救援，随后这名滑雪者被带上了雪地救生船。　　据知情者介绍，出事的地点在万龙雪场的五彩龙道，当时这名女子应该是正要转弯，但是五彩龙道十分狭窄，雪道的两侧就是树林，这名女子可能是由于速度太快才撞上了雪道旁边的树木。　　春节期间，本报记者曾对万龙雪场做过探访，万龙雪场雪道两侧确实存在不少树林，而且部分树林与雪道之间是没有防护网的，而这正是万龙雪场吸引人的地方。这些面积不大的小树林足以让一些滑雪者体验一把“林海雪原”的感觉。　　万龙雪场负责安保的曹经理对记者说，部分不适宜滑野雪的小树林雪场也加装了防护网，但是经常出现被人为破坏的现象。雪场的人手有限，每次巡逻时只要发现就会及时修补，另外一些没有加装护网的树林，巡逻队每天会对里面的碎石、灌木进行检查，并在夏天进行集中的清理。　　业内说法 很多人对“滑雪好”存在错误认识　　1 月16日，一名北大的女研究生在崇礼县万龙雪场滑雪时，出现意外身亡。据目击者描述，当时这名女生一头扎在雪里，身边都是树皮。三天后，一名10岁的北京 男孩在太舞滑雪时跌下山崖，在送往医院之后经抢救无效不幸身亡。记者从男童的父亲关先生处了解到，孩子跌下位置附近的雪道并不陡峭，但是雪道两侧的挡风板 下方却有着高度在一米左右的缺口。　　法晚记者曾在事发后前往崇礼的太舞、万龙滑雪场，探访雪道两侧的防护网情况，以及雪场的相关防护措施。记者发现，这两个雪场的雪道两侧均已加装防护网。　　魔法滑雪学院的校长张岩此前接受本报记者采访时表示，目前很多滑雪者最大的问题就是不清楚自己的滑雪水平可以上哪一条雪道，对“滑雪好”的概念存在错误认识。　　张岩还指出，现如今有很多的滑雪联赛，其中有不少选手基本功不够，“基本功不扎实，主要体现在无法很好地控制速度，出现突发情况不能很好地处理。”聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
