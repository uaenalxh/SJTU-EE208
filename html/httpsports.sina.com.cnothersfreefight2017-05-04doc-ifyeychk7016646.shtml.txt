赛法斗70公斤级八大高手 赵玉飞或与宝力高弟子PK_武术散打_新浪竞技风暴_新浪网
2017-05-04
赛法斗70公斤级八大高手 赵玉飞或与宝力高弟子PK　　提起“拼命老爸”赵玉飞，相信大家都不陌生，谁都知道这位“战无极”擂台上的人气明星拼到腿瘸、一瘸一拐发表夺冠感言的故事！汽修学徒、叉车司机、饭店服务员、保安、工地小工……无论工作如何变换，赵玉飞对武术的执着追求始终不变，日复一日、年复一年坚持苦练技术，并向女儿证明通过不断努力，梦想可以实现。　　赵玉飞还在自己开办的战无极散打健身会馆教女儿练习搏击，帮她提高身体素质。赵玉飞透露，本来锻炼身体这个硬性规定没有商量余地，但女儿比较会说话，有时会跟他商量着怎么不训练！只要训练量达到，他也会让着、随着女儿。　　时隔10个多月，即将又能在“战无极”的擂台上，看到“拼命老爸”的身影了！5月8-13日，赵玉飞将参赛在北京体育大学举行的第二届全国体育院校功夫全能技大奖赛。再次去北京比赛，赵玉飞抑制不住内心的喜悦，他说：“真的很开心，很开心，有很多老朋友许久未见了！从“我就是拳王”上海海选现场一路走到北京总决赛擂台，大家都结下了深厚的友谊。再次去北京参赛，就像回到一个大家庭中，有很多老朋友可以一起交流学习”。　　赵玉飞本次参加功夫站立技（赛法斗）男子70公斤级比赛，飞哥所处的局势如何呢？待小编给大家分析一下。站立技该级别男子组共8人报名，选手均有比赛经验，谁都不是一张白纸！除了赵玉飞是80后，其余7人全部为90后，年龄介于18-26岁。　　赵玉飞身高位居第二，在“战无极”擂台上的经验最老道，战绩为6胜1负，在“我就是拳王”的擂台上4连胜，TKO秦红军、贡巴太，通过判定战胜符厚龙、刘仁贵，最后获得全国总决赛冠军；在拳王统一战的擂台上，通过断头台降服体院高手胡景辉、刘梦雨。遗憾的是他在中法对抗赛中，未能实现战无极7连胜，遭塞利姆十字固降服，手臂受伤。　　飞哥表示去年手臂受伤后，自己主要是请教医生做一些拉伸动作慢慢恢复，前不久在训练中又受伤了，故不能报名参赛功夫全能技了。之前没参加过赛法斗，所以想在这个项目上尝试一下。对于没有接触过的赛事，赵玉飞认为熟读规则，进行针对性的训练最重要。　　有趣的是，刘仁贵此次也未报名参赛功夫全能技，而是报了站立技。这位私人保镖的鞭腿不容小觑，曾在“我就是拳王”全国总决赛中踢瘸拼命老爸，与冠军宝座失之交臂！此次70公斤级站立赛，两人有可能再次相遇。对于二番战，赵玉飞说：“很高兴阿贵也能来参加这次比赛，经过上一次比赛，我俩也是好朋友了。经过这么久的磨练，都成长了，希望他能取得好成绩。如果再次在擂台上相遇，我想他和我一样都会全力以赴”。　　除了赵玉飞、刘仁贵、李军，其余5位选手均是第一次参加“战无极”比赛。刘仁贵在“战无极”擂台的战绩为3胜2负1平，李军在中意拳王对抗赛中遭南多-加斯达KO。　　根据博克森竞赛部提供的报名资料，未参赛过“战无极”的5人中有3人是散打锦标赛冠军，另外2人打过商业赛，成绩不错。其中，来自宝力高搏击训练基地的吕晓东蝉联2010-2014年五届内蒙古散打锦标赛70公斤级冠军，他还是2014年内蒙古第十三届运动会70公斤级冠军，2015年首届国际综合武术搏击争霸赛冠军。道行不浅呐！看来吕晓东也是本次功夫站立技比赛夺冠热门人选。　　由此可见，男子70公斤级功夫站立技比赛定会紧张刺激、精彩纷呈！赛事将在中国互联网电视全程播出，敬请期待。聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
