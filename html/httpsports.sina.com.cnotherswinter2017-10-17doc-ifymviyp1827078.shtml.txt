冰雪器材装备国产化须“弯道超车”_冰雪_新浪竞技风暴_新浪网
2017-10-17
冰雪器材装备国产化须“弯道超车”　　在三亿人参与冰雪运动和2022年冬奥会的双重利好推动下，我国冰雪产业快速发展，冰雪产业市场容量急速扩充。而在赚钱效应最显著的冰雪器材装备领域，国外品牌和产品占据绝大数份额，如何提升“中国制造”的技术水平和市场份额，成为摆在中国企业面前的一道必答题。从近期一些企业的举措上看，利用自身在本土的渠道优势，结合国外品牌的技术优势，合资创立基于中国市场的品牌，建立面向中国市场的产品研发和生产体系，正在成为一种现象，我们姑且将之视作特殊市场环境下的“弯道超车”，这种弯道超车可以在短期内实现冰雪器材装备国产化进程，提升中国企业在该领域里的市场竞争力。　　就中国冰雪器材装备国产化这一课题，国家体育总局也正联合国家工业和信息化部等有关部门，一道着力破解。参与到此进程的工信部机械工业信息研究院副院长石勇认为，在精密技术和材料学上都要求极其高的冰雪器材装备制造业领域，中国和发达国家确实存在不小差距，“有些领域我们可以组织专家进行技术攻关，但在全球经济一体化的今天，我们还需要借助外力，和国外同行进行合作，以节省时间成本。” 中国体育用品业联合会秘书长罗杰在接受本报记者采访时也曾指出，冰雪产业装备器材制造业的发展，需要选择合适的领域和项目，方式上可以依托目前有实力的体育用品企业，采取收购、合资等方式，引进国外成熟技术，进行短平快的国产化进程。　　一些地方已经看到该领域蕴藏的巨大市场红利，开始利用政策导向去引领企业借助外力涉足冰雪装备的国产化进程。2022年冬奥会承办地张家口所在的河北省政府办公厅出台的《关于支持冰雪运动和冰雪产业发展的实施意见》中，为提升冰雪器材装备研发制造能力，提出建设冰雪装备制造聚集区，鼓励企业通过海外并购、合资合作、联合开发等方式，引进国内外高端冰雪装备制造企业。冰雪运动传统强省黑龙江制定了《关于加快我省冰雪旅游产业发展的实施意见》中也同样指出，促进装备制造业与冰雪旅游融合发展，依托哈尔滨、齐齐哈尔、牡丹江等地现有制造加工生产能力，积极引入国外先进冰雪旅游装备制造企业，合资合作研发制造索道、压雪机、造雪机、制冰机、雪地摩托等冰雪设施装备，打造核心品牌，巩固黑龙江在中国冰雪旅游产业的龙头地位。　　在政策红利下，国内一部分冰雪行业内的企业已经先行试水，转换思路，与国外企业合作推进冰雪装备制造业的国产化发展。世纪星冰雪多年深耕冰雪领域，涉足青少年培训、赛事运营、场馆运营、冰上演艺娱乐，同时也开始涉足冰雪运动硬件市场，日前世纪星冰雪与美国Sport Ice合作，将在国内引进Sport Ice清冰车技术并国产化，利用其节能环保、节省空间的优势，将为人们提供更好的体验环境。这也是一次中外企业合作的范例，利用最新最好的设备、资源、技术支持，借助国际品牌，运用中国制造，实现冰雪产业大型设备的国产化，弥补中国冰雪产业快速发展期过程中硬件设施的不足。世纪星董事长范军表示，未来几年世纪星还将积极参与国内外优秀冰雪项目的并购和收购，博采众长，积极发挥世纪星冰雪在冰雪行业中的领军作用。　　除了冰雪产业领域内的企业外，一些产品相近的企业也开始向冰雪器材装备领域靠拢。上市公司雪人股份主营制冷系统和冷链物流系统，该公司去年开始通过收购瑞典、意大利的造雪系统品牌，结合自身技术，研发更适合中国雪场应用的集成造雪系统，其产品在造雪性能和节能上具有突出优势。在今年年初，大型国企中煤集团与法国MND集团合资成立安美地（张家口）山地发展有限公司，将其滑雪装备制造项目落子张家口。MND集团曾是索契冬奥会的索道供应商，主要经营范围涉及景区开发和索道、造雪机、娱乐设备、安全设施、竞赛设备制造等领域。与中煤集团的合作，MND集团部分专业的冰雪装备产品在将张家口生实现生产和制造、组装。据了解，公司计划到2022年前生产50条索道，预计今年年底生产出样机。承接制造项目的中煤集团子公司张家口煤矿机械公司主要生产矿用链条，拥有强大的装备制造能力，借机利用MND集团优秀的技术优势，助力我国冰雪装备制造业实现国产化发展驶入快车道。　　中国体育报聚焦 |  |  |  |  |  |  |  | Copyright © 1996-2017 SINA Corporation, All Rights Reserved新浪公司
